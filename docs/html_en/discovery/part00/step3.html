Step 3. Organizing the dashboard¶

The final step is to create a chart widget, text widget, and filter widget to fill the empty dashboard. The dashboard can be edited in the following order.

Using the 판매현황 데이터 created earlier, let’s add a key performance indicator chart and a linear chart to the dashboard.

In the empty dashboard, click the Chart button to create a chart.

Creating a key performance indicator chart¶

The first chart you will be creating is a key performance indicator (KPI) chart. The key performance indicator chart is a simple yet powerful chart that displays the goal of the organization in an intuitive manner. The goal of our dashboard is to clearly present sales data. As such, the key performance indicator chart should include total sales, sales forecast, and total profit. What should we do? Simply click the three measurement columns named “Sales,” “SalesForecast,” and “Profit” under the data menu. This task is called pivoting. The pivoted columns are automatically summed and placed in the panel. Once the columns are in the panel, suitable charts are recommended. How about clicking the recommended key performance indicator chart?

The key performance indicator chart is created as follows. To make it more presentable, let’s enter the chart properties menu on the right.

Click to enter the Common Settings panel and add icons to each measurement column.

Click to enter the number format panel and change the decimal point display and abbreviation display.

The most important feature of the key performance indicator chart is comparing present achievements with past performance. Click to enter the Complementary Indicator Settings panel. Set a complementary indicator, and check the % improvement in performance compared to the previous month. If you wish, you can emphasize the complementary indicator instead of the original indicator.

Click Save to display the chart in the dashboard.

Creating a linear chart¶

Next, let’s create a linear chart, the most basic type of chart. Shall we take a look at how sales and profit change over time? Again, click the Chart button to begin drawing a new chart. Click the OrderDate, Profit, and Sales columns to see how the values change over time. Click the recommended linear chart.

A linear chart is drawn. Open the chart properties window, and change the line shape to “round.”

There is too much data as the unit of collection of OrderDate is time. To view by month, go to the menu of the OrderDate column, and set Granularity as Month. The entire data is displayed. Click Mini Map on the upper right to remove the mini map from the chart.

Click on the right menu, and change colors using the Color Settings panel.

Click Save, and drag and drop the chart to the desired position. Add information to the dashboard by adding a text widget. Click Done to end dashboard editing.

In this tutorial, you learned how to draw two chart types. Using the interactive dashboard, you can select a chart or add filters to present data as desired. You can also modify the chart, or add/delete data if required.
