Creating a dataset¶

A dataset, which is the basic unit of data preparation, refers to the entity that is the target of data operations. The two types are: Imported Dataset and Wrangled Dataset.

  • Imported Dataset: A source data entity before the implementation of transformation rules

  • Wrangled Dataset: Wranlged A data entity that becomes the target of analysis following the implementation of transformation rules

A Wrangled Dataset is created during the dataflow designation process, which defines the transformation rules, while an Imported Dataset is created during the main procedure.

From the left panel of the dataset menu, go to MANAGEMENT>**Data Preparation**>**Dataset**.

Next, on the upper right of the dataset page, click the +Create new dataset button to create a new dataset.

In the dataset creation page, select the dataset type.

  • My file: Create a dataset by opening a user file or via the URI (upcoming feature) (See Creating a file-type dataset for detailed procedures).

  • Database: Create a dataset using external database access information and queries (See Creating a database-type dataset for detailed procedures).

  • Staging DB: Create a dataset based on the staging DB configuration used to run Metatron (See Creating a Staging DB dataset for detailed procedures).

Note

The Staging DB is an in-cluster database that stores data temporarily in order to facilitate data loading. Hive is generally used for it.

Creating a file-type dataset¶

Create a dataset by opening a user or via the URI (upcoming feature).

  1. On the data type selection page, select File.

  2. Select a file to be used as a data source from your local PC. You can click the Import button to select a file, or drag and drop the file into the box. After importing the file, click the Next button.

  3. Check the grid of the uploaded file, and designate a column separator. Proceed if the data is successfully displayed.

  4. Enter the name and description of the dataset, and click the Done button.

  5. Once the dataset is created, the dataset list is displayed. You can check that the list contains the newly created dataset.

Creating a database-type dataset¶

Create a dataset using external database access information and queries.

To create a database-type dataset, you should first create a data connection. See Creating a data connection for detailed procedures.

After establishing the data connection, go to MANAGEMENT>**Data Preparation**>**Dataset** > +Create new dataset.

  1. On the data type selection page, select Database.

  2. Select the data connection, and press the Test button to check that the connection is valid.

  3. Select the data. You can either select a table from the connected database, or write a query yourself.

    • Table: Check the data to be stored after selecting the database and table name, and click the Next button.

    • Query: Write a query to retrieve the desired data, and click Run to display the data below. After checking the data, click the Next button.

  4. Enter the name and description of the dataset, and click the Done button.

  5. Once the dataset is created, the dataset list is displayed. You can check that the list contains the newly created dataset.

Creating a Staging DB dataset¶

Create a dataset based on the staging DB configuration used to run Metatron.

The Staging DB dataset is the same as the database-type dataset, but does not involve any designation of data connections.

  1. On the data type selection page, select Staging DB.

  2. Select the data. You can either select a table from the connected database, or write a query yourself.

    • Table: Check the data to be stored after selecting the database and table name, and click the Next button.

    • Query: Write a query to retrieve the desired data, and click Run to display the data below. After checking the data, click the Next button.

  3. Enter the name and description of the dataset, and click the Done button.

  4. Once the dataset is created, the dataset list is displayed. You can check that the list contains the newly created dataset.
