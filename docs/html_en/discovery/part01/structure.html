Components of Metatron Discovery¶

Metatron Discovery retrieves data from data sources loaded into the server or other external sources, performs advanced analytics, and outputs results in the form of charts and reports. To utilize this system, you must understand its overall structure shown below:

Data Preparation¶

Data preparation refines data from data sources to be loaded into Metatron. See Data preparation for more details on data preparation.

Data Storage¶

Data storage manages data loaded into the Metatron engine for analysis and visualization. See Data Management for more details on the data management function.

Data Analysis and Visualization¶

Each module below allows users to perform visualization-based search and analysis of stored data.

Workspace¶

Workspace can be managed according to user access to workbook, workbench, and notebook within an organization. See Workspace for more details on the workspace function.

Workbook, Dashboard, Chart¶

Workbook is a module that supports sharing and presentation of dashboards and charts. See Workbook for more details on the workbook function.

Notebook¶

In notebook, you can perform advanced analytics based on machine learning. See Notebook for more details on the notebook function.

Workbench¶

Enables SQL data analytics. See Workbench for more details on the workbench function.

Data monitoring¶

This function monitors data use based on data query statistics and query logs. See Data Monitoring for more details on the data monitoring function.

User permission and account management¶

You can add/delete users or manage user permission.
