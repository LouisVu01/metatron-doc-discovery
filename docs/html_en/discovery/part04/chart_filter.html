Chart Filters¶

A chart filter defines what range of data is to be shown on the chart. This chapter describes how to set up and make use of chart filters.

Automatically included filters¶

The following column filters are included automatically when a chart is created:

  • Timestamp column filter: As a time-series data store, Metatron Druid necessarily uses a time filter.

  • Recommended filters: Column filters designated as &ldquo;recommended filters&rdquo; during the registration of the data source.

  • Dashboard with global filters applied: Filters applied to all charts registered in the dashboard.

Chart filter panel¶

There is a chart filter panel located on the right-hand side of the chart home screen. On this panel, you can easily view and configure registered filters.

  1. Filter number: The number of charts to which a filter is currently applied is shown next to the name of the chart panel.

  2. Add/edit filter: Pushing + at the top right either adds a new filter or opens a pop-up for configuring an existing filter.

  3. Columns with filters: The top part of individual filters displays information about the column to which the filter is applied.

  4. Detailed filter settings: Pushing the hamburger menu at the top right of individual filters allows you either to reset the filter or configure the details of the filter.

Chart filter dialog box¶

Click the button at the top of the chart filter panel or click the button in each filter area to open the chart filter dialog box. With this dialog box, you can add a new filter or configure an existing filter.

Dimension filtering¶

From the connected data source, select a dimension on which to create a filter.

  • Value range: Select whether to filter the chart by a single or multiple data categories.

    • Single: Select one data category by which to filter the chart.

    • Multiple: Select multiple data categories by which to filter the chart.

  • Search: If there are too many column attributes, this function allows you to limit the results only to those you wish to see.

    • Search by name: Search with the name of a column attribute.

    • Attribute filtering: Filters attributes either by matching attribute names with regular expressions or wildcards, or by applying conditions based on the measure range.

  • Defined value: Used to add &mdash; as a filter criterion &mdash; an attribute that is not contained in the column. This function creates filters by predicting data which, although not currently found in the data source, is likely to be entered in the future.

Timestamp column filter settings¶

Dimensions with time icons displayed are a type of timestamp for which a timestamp filter can be configured. Although they are set to &ldquo;All Time&rdquo; by default, you can select Relative or Specific if you wish to display only data from a certain period in the chart.

&ldquo;Relative&rdquo; sets a period of time relative to the present and displays only data from the applicable period of time in the chart.

&ldquo;Specific&rdquo; directly sets a certain period of time of data and displays only data from the applicable period of time in the chart.

Measure filtering¶

From the connected data source, select a measure on which to create a filter.

Once you have selected the measures to be set as filters, designate the range of values to filter.
