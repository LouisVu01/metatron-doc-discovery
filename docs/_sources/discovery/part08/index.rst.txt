계정 관리
---------------------------------------

관리자는 Metatron Discovery 사용자들의 멤버쉽과 사용 권한을 설정·관리할 수 있으며, **그룹** 기능을 이용하여 이러한 관리를 더욱 효율화할 수 있습니다.

사용자 관리를 위해서는 메인 화면 좌측 패널에서 :menuselection:`ADMINISTRATION --> 사용자` 클릭 후 원하는 하위 메뉴를 선택하면 됩니다.
